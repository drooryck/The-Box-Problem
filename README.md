# The Box Problem

This repository contains a writeup and supporting code for an investigation of **the Box Problem**: a version of the secretary problem from decision theory. The repo explores the problem analytically and computationally, deriving some theory and backing it up with simulation for some proposed strategies.

## Report

**[View the written report](https://drooryck.github.io/The-Box-Problem/)**

## Repository contents

* `written report/` — the final LaTeX/PDF report
* `uniform distribution/` — experiments using a uniform random model
* `normal distribution/` — experiments using a normal random model
* `simple simulation of game/` — simulation code for the basic game
* `simple chance theories/` and `theories/` — exploratory approaches and candidate models
